# NEURØ Mascot Blueprint Manifest

## Technical Specifications
- **Model Name**: NEURØ v2.5 (Stable)
- **Height**: 120.5mm
- **Width**: 85.2mm
- **Depth**: 64.0mm
- **Weight**: ~142g (Standard PLA)

## Material Requirements
- **Primary Shell**: Reinforced Polymer (PLA+ or PETG)
- **Visor Component**: Translucent Resin (SLA) or Clear PETG
- **Joint Type**: Ball-and-Socket (Print-in-place or Assembly)

## 3D Printing Parameters
- **Layer Height**: 0.12mm (High Detail)
- **Infill**: 15% Gyroid (Structural Integrity)
- **Supports**: Required for shoulder overhangs and visor cavity
- **Adhesion**: Brim recommended for resin prints

## Assembly Instructions
1. Print the primary chassis in matte black or dark gray.
2. Print the visor in a transparent material.
3. Insert the LED core (optional) into the chest cavity before sealing the back panel.
4. Snap-fit the limbs into the ball joints.
